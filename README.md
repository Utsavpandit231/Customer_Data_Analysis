# Customer_Data_Analysis
🛍️ Customer Shopping Data Analysis
📌 Project Overview

This project focuses on analyzing customer shopping behavior data to uncover meaningful business insights using Python, PostgreSQL, SQL, and Power BI.

The project follows an end-to-end data analysis workflow including:

Data Cleaning
Data Transformation
SQL Analysis
Business Insight Generation
Interactive Dashboard Visualization

The objective is to help businesses understand customer purchasing patterns, spending behavior, product performance, and customer segmentation for better decision-making.
